  • Name: Pass the Bomb - An electronic game

  • Year: Dec 2019

An electronic game using a 16-Bit ALU

We looked up popular multiplayer games and try to think of variations of them. Eventually, we settled on the game “Pass the Bomb”. This game idea is adapted from a kids’ party game that we all have played, where a ball is passed around and the loser is the player holding the ball when the timer ends.

Rules:
1. You can pass the bomb to the person left, right or opposite of you
2. The longer you hold the bomb, the more points you get
3. If you hold the bomb for more than 3 seconds, you will get blown up
4. If you are holding the bomb when the game ends (random time interval), you will get blown up
5. The person with the highest points is the winner
6. The game is played on a game board and is a strategy game, where players can gain points based on how long they hold the bomb for.

End Game
The player who gets blown up should lose even if he has the highest score
We disregard the score of the player who gets blown up/loses the game

Draw Game
Happens when 2 or more players(who did not get blown up) end up with the same highest scores
The code in Mojo compares the score of 3 players (excluding the player who got blown up by the bomb)
Using the highest score out of the 3 players, if more than 1 player shares the highest score, mojo goes to draw state.
